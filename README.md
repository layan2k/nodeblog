# Node Express Blog

Foobar is a Python library for dealing with word pluralization.

## Installation

Use the package manager [npm](https://www.npmjs.com/package/npm) to install the file dependencies.

```bash
npm install
```

## Usage

```shell

# To open Express Server on Local Host
npm start

```
